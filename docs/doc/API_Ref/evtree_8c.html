§ evt_find_ent_list()

int evt_find_ent_list ( struct evt_context *  tcx,
enum evt_find_opc  find_opc,
struct evt_rect *  rect,
struct evt_entry_list *  ent_list 
)

Find all versioned extents which intercept with the input one rect. It attaches all found extents and their data pointers on ent_list if no_overlap is false, otherwise returns error if there is any overlapped extent.

Parameters
rect[IN] Rectangle to check.
no_overlap[IN] Returns error if rect overlap with any existent extent.
ent_list[OUT] The returned entries for overlapped extents.
