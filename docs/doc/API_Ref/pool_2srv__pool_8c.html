Create a (combined) pool(/container) service. This method shall be called on a single storage node in the pool. "target_uuids" shall be an array of the target UUIDs returned by the ds_pool_create() calls.

Parameters
[in]pool_uuidpool UUID
[in]uidpool UID
[in]gidpool GID
[in]modepool mode
[in]ntargetsnumber of targets in the pool
[in]target_uuidsarray of ntargets target UUIDs
[in]groupcrt group ID (unused now)
[in]target_addrslist of ntargets target ranks
[in]ndomainsnumber of domains the pool spans over
[in]domainsserialized domain tree
[in,out]svc_addrssvc_addrs.rl_nr inputs how many replicas shall be created; returns the list of pool service replica ranks
