Enumerator
DAOS_EVF_NO_POLL 

An event will be queued in EQ on completion and wait for polling by default. With this flag, the completed event will not be queued in the EQ anymore, upper level stack is supposed to be notified by callback.

DAOS_EVF_NEED_LAUNCH 

Only useful for parent event: without this flag, a parent event will be automatially launched if any child event is launched. With this flag, a parent event always needs to be explicitly launched.

Parameters

§ dc_task_schedule()

int dc_task_schedule ( tse_task_t *  task,
bool  instant 
)

Schedule the task created by dc_task_create, this function will execute the body function instantly if instant is true.

Schedule task created by dc_task_create_ev(), if the associated event of task is the private event, this function will wait until completion of the task, otherwise it returns immediately and its completion will be found by testing event or polling on EQ.

The task will be executed immediately if instant is true.
