§ dc_pool_query()

int dc_pool_query ( tse_task_t *  task)

Query the latest pool information (i.e., mainly the pool map). This is meant to be an "uneventful" interface; callers wishing to play with events shall do so with cb and cb_arg.

Parameters
[in]poolpool handle object
[in]ctxRPC context
[out]tgtsif not NULL, pool target ranks returned on success
[out]infoif not NULL, pool information returned on success
[in]cbcallback called only on success
[in]cb_argargument passed to cb
Returns
zero or error
