Detailed Description

Stackable Module API Provides a modular interface to load and register server-side code on demand. A module is composed of:

  • a set of request handlers which are registered when the module is loaded.
  • a server-side API (see header files suffixed by "_srv") used for inter-module direct calls.

For now, all loaded modules are assumed to be trustful, but sandboxes can be implemented in the future.
