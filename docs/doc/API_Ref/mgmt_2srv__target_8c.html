§ ds_mgmt_tgt_pool_iterate()

int ds_mgmt_tgt_pool_iterate ( int(*)(const uuid_t uuid, void *arg)  cb,
void *  arg 
)

Iterate pools that have targets on this node by scanning the storage. cb will be called with the UUID of each pool. When cb returns an rc,

  • if rc == 0, the iteration continues;
  • if rc == 1, the iteration stops and returns 0;
  • otherwise, the iteration stops and returns rc.
Parameters
[in]cbcallback called for each pool
[in]argargument passed to each cb call
