§ evt_iter_probe()

int evt_iter_probe ( daos_handle_t  ih,
enum evt_iter_opc  opc,
struct evt_rect *  rect,
daos_hash_out_t *  anchor 
)

Based on the opc, this function can do various things:

  • set the cursor of the iterator to the first extent in the evtree.
  • find the provided extent or iteration anchor.

This function must be called after evt_iter_prepare, it can be called for arbitrary times for the same iterator.

Parameters
opc[IN] Probe opcode, see evt_iter_opc for the details.
rect[IN] The extent to probe, it will be ignored if opc is EVT_PROBE_FIRST.
anchor[IN] The anchor to probe, it will be ignored if rect is provided.
