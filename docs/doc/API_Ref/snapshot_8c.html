Destroy a snapshot. The epoch corresponding to the snapshot is not discarded, but may be aggregated.

Parameters
coh[IN] Container handle
epoch[IN] Epoch of snapshot to destroy
ev[IN] Completion event, it is optional and can be NULL. The function will run in blocking mode if ev is NULL.
